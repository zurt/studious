# Studious - Roadmap

## MVP (Current)

- [x] PDF and image upload
- [x] PDF page rendering to PNG
- [x] Tesseract OCR integration (Japanese horizontal + vertical)
- [x] Anthropic Claude VLM integration with custom prompt
- [x] Side-by-side page image + transcription viewer
- [x] Async job queue with SSE progress streaming
- [x] Configurable provider selection and model choice
- [x] Structured logging with correlation IDs
- [x] Supply chain security (package manager cooldown)
- [x] Quality benchmarking framework

## Planned Features

### Near-term
- [ ] Furigana display — render furigana above kanji in transcriptions
- [ ] Vocabulary lookup — click a word to see dictionary definitions
- [ ] LLM translation — translate transcribed text with grammar breakdown

### Medium-term
- [ ] Batch document processing — queue multiple documents at once
- [ ] Search across transcriptions — full-text search over all transcribed content
- [ ] Export — download transcriptions as markdown, PDF, or Anki cards
- [ ] Multiple VLM providers — support OpenAI, Google Gemini alongside Anthropic

### Long-term
- [ ] Spaced repetition — track vocabulary review with SRS scheduling
- [ ] Reading progress — track pages read, time spent, vocabulary encountered
- [ ] Collaborative study — share documents and annotations between users
- [ ] Mobile-friendly UI — responsive layout for tablet/phone use
